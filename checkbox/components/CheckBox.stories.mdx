# Check Box
___
`Checkbox` will serve as generic checkbox for kaios app.

## Usage

> Props that can be passed are:

<Props of={CheckBox}/>

Checkbox can have four states-
1. Unchecked
1. Checked
1. Uncheck Disabled
1. Checked Disabled
